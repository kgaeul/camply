
## Camply
캠핑 전문 예약 사이트
- 캠핑장 예약과 캠핑용품 구매를 한번에 할 수 있는 사이트 제작 
- 타 사이트와 달리 캠핑장을 전문적으로 다루며 캠핑장 유형에 따라 캠핑을 나눠서 정보를 제공하는 사이트 제작

### 프로젝트 소개
![image](https://github.com/kgaeul/camply/assets/143372208/ce5db1f7-79fe-4b0f-9a49-a560af8397ae)
#### 메인페이지
사이트의 메인화면으로 검색창으로 이동이 가능하며 캠핑장 유형에 따른 검색이 가능하다.
<hr>

![image](https://github.com/kgaeul/camply/assets/143372208/eb97aa91-c7c4-415c-beaa-1fb3a867d788)
#### 검색창 페이지
지역, 이름, 인원수 등의 세부 옵션과 함께 검색이 가능하다.

![image](https://github.com/kgaeul/camply/assets/143372208/28bbc11f-fb96-4628-83f3-c26401f0d06f)
#### 검색 목록 페이지
검색 결과 페이지로, 캠핑 유형별 검색이나, 검색페이지에서 검색한 결과를 보여준다.
이미지 클릭 시 상세페이지로 이동한다.
